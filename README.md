# finance-expenses-dashboard-excel
Interactive Finance Expenses Dashboard built in Excel using Pivot Tables, Charts, and Slicers to analyze income, expenses, and spending patterns.

📊 Finance Expenses Dashboard (Excel)
📌 Overview

This project is an interactive Finance Expenses Dashboard built using Microsoft Excel to track and analyze income and expenses.
It provides clear insights into spending behavior using dynamic charts and filters.

🚀 Features

📈 Income vs Expenses comparison

📊 Category-wise expense breakdown

📅 Monthly trend analysis

🧾 Sub-category insights (Rent, Groceries, Salary, etc.)

🎛️ Interactive slicers for filtering

🎨 Clean and user-friendly dashboard design

🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables & Pivot Charts
Slicers
Data Cleaning
